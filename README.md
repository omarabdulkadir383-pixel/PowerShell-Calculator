
# Colorful PowerShell Calculator

A modern, colorful calculator built entirely in PowerShell that provides a clean command-line interface with an improved user experience through color-coded menus, prompts, and results.

This project demonstrates how PowerShell can be used to build interactive console applications while applying programming fundamentals such as functions, conditional statements, loops, input validation, and user-friendly interface design.

## Features

* Color-coded user interface
* Interactive command-line menu
* Basic arithmetic operations

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Input validation
* Error handling for invalid entries and division by zero
* Easy-to-read output
* Beginner-friendly PowerShell project

## Technologies Used

* PowerShell
* Windows Terminal / PowerShell Console

## Learning Objectives

This project was created to strengthen PowerShell scripting skills by practicing:

* Functions
* Variables
* Loops
* Conditional logic
* User input handling
* Error handling
* Console formatting with colors
* Building interactive CLI applications
